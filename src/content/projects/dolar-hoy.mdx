## El proyecto

Dolar Hoy nació con un objetivo ambicioso: construir el portal de referencia para el tipo de cambio en Perú, diferenciándose de competidores como cuantoestaeldolar.pe mediante un mapa interactivo que muestra casas de cambio verificadas cerca de la ubicación del usuario, datos actualizados cada 30 segundos y un directorio estructurado con información de la SBS.

## Mi rol

Me incorporé cuando el proyecto ya tenía una base funcional. Mi participación fue en la **mejora de secciones existentes, creación de nuevas páginas y soporte en la configuración de infraestructura**. El proyecto actualmente está en pausa.

## Lo que aporté

<CaseSection title="Aportes técnicos" eyebrow="Implementación" tone="accent">

### Desarrollo frontend con Astro + React Islands

La arquitectura usaba Astro como base SSG con componentes React hidratados selectivamente usando la directiva `client:*` (Astro Islands). Me encargué de mejorar las secciones informativas y crear páginas nuevas respetando esta arquitectura para no comprometer el rendimiento del sitio.

### Sección de noticias y blog

Configuré e implementé las secciones de **noticias** y **blog** del portal, incluyendo la estructura de rutas, los layouts de artículo y la integración con el sistema de contenido.

### Infraestructura: primer contacto con AWS y DevOps

En este proyecto tuve mi primera exposición práctica a herramientas de infraestructura:

- **Consola SSH en instancia AWS EC2:** accedí y ejecuté comandos directamente en el servidor del proyecto.
- **Portainer:** actualicé imágenes Docker y stacks del entorno de producción desde la interfaz web de Portainer.
- **Caddy como reverse proxy:** creé un nuevo stack de Caddy y configuré el `Caddyfile` desde consola SSH para que el dominio renderizara correctamente los datos de la aplicación.
- **Variables de entorno:** actualicé los envs del sistema junto a los stacks e imágenes para sincronizar la configuración del dominio con los servicios internos.

</CaseSection>
