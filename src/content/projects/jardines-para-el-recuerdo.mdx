## El proyecto

Jardines para el Recuerdo fue la primera web desarrollada para una asesora independiente de Parque del Recuerdo. El sitio operaba en el mismo nicho que su sucesor: conectar a familias con opciones de sepulturas, nichos, mausoleos y planes de prevención en camposantos de Lima.

## Lo que ocurrió

<CaseSection title="Transición de marca" eyebrow="Contexto crítico" tone="accent">

La empresa Parque del Recuerdo solicitó a la clienta que retirara la web por motivos de uso de marca. Dado el plazo extremadamente corto para bajar el sitio, no fue posible aplicar redireccionamientos 301 para preservar el posicionamiento orgánico alcanzado.

Este proyecto fue el antecedente directo de **Asesora de Sepelio**, donde se reconstruyó todo desde cero: nueva identidad visual, nuevo nombre, nueva arquitectura de contenido y nueva estrategia SEO.

</CaseSection>

## Lo que este proyecto representa

<CaseSection title="Aprendizaje aplicado" eyebrow="Resiliencia" tone="default">

La experiencia de gestionar una transición de marca bajo presión de tiempo, donde la prioridad fue garantizar que la clienta tuviera presencia online operativa en el menor tiempo posible, aun perdiendo el histórico de posicionamiento del dominio anterior.
